# 操作系统原理 — 题库生成参考

本文档记录了操作系统原理课程中常见的考点、易错点和答案补全经验。处理类似 OS 课程时可复用。

## 答案补全规则

当 `assignments_md/*.questions.json` 中大量题目答案缺失时（常见于未提交的章节测试）：

### 判断题推断规则
- 含"所有/一定/必须/只能/都是/必定"等绝对化表述 → **大概率错**
- 含"不一定/也可能/有时"等缓和表述 → **大概率对**
- 判断定义的正误 → 参照教材标准表述
- 判断两个概念的关系 → 注意区分包含/被包含/等价

### 填空题答案模式
- CPU状态：目态/管态（目态=用户态，管态=核心态）
- OS类型：分时/实时/批处理
- 信号量计算：|S|=阻塞进程数，S<0→阻塞
- 死锁：四个必要条件（互斥/请求保持/不可剥夺/循环等待）
- 存储管理：地址重定位（静态/动态）、局部性原理、页号/段号
- 设备管理：寻道时间/DMA/虚拟设备/块设备/设备无关性

## 章节核心知识点速查

### 第1章 操作系统引论
- OS定义：硬件之上第一层软件
- OS目标：方便性 > 有效性 > 可扩充性 > 开放性
- OS作用：接口（用户） + 资源管理者（系统） + 抽象（虚拟计算机）
- 三种接口：命令接口、程序接口（系统调用）、图形接口
- OS类型：批处理（无交互）、分时（交互，时间片轮转）、实时（实时性+可靠性优先）
- CPU状态：目态（用户态，不可执行特权指令）、管态（核心态，可执行所有指令）
- 多道程序设计：内存中同时存放多道程序，宏观并行微观串行
- OS性能分类：单用户单任务（MS-DOS）、单用户多任务（Windows）、多用户多任务（UNIX/Linux）

### 第2章 处理机管理（进程）
- 进程定义：程序的一次执行，动态（有PCB），资源分配和调度基本单位
- 进程三态：就绪→运行→阻塞。阻塞不能直接运行！
- PCB：OS感知进程存在的唯一标志
- 同步vs互斥：同步=直接制约（合作），互斥=间接制约（资源竞争）
- 临界区：访问临界资源的代码段
- PV操作：P(S)=S-1,若S<0阻塞；V(S)=S+1,若S≤0唤醒。互斥初值=1，同步初值=0/N
- 死锁四条件：互斥、请求保持、不可剥夺、循环等待
- 线程：TCB、同进程内线程可并发、引入内核线程后CPU调度以线程为单位
- 原语：不可中断的原子操作

### 第3章 处理机调度与死锁
- 三级调度：高级（作业调度，后备→内存）、中级（挂起/激活）、低级（进程调度，就绪→CPU）
- 调度算法：FCFS（非抢占）、SJF（平均周转最短）、HRRN（响应比=(等+运)/运）、RR（时间片）、多级反馈队列（综合最优）
- 死锁处理：预防（破坏条件）、避免（银行家算法）、检测（资源分配图）、解除（撤销/剥夺）
- 安全状态≠死锁：安全→无死锁，不安全→可能死锁，死锁⊂不安全
- 最少资源数公式：n×(w-1)+1

### 第4章 存储管理
- 地址重定位：逻辑地址→物理地址。静态=装入时，动态=执行时
- 分区分配算法：首次适应（地址递增）、最佳适应（大小递增，挑最小）、最差适应（大小递减，挑最大）
- 分页：物理单位、大小固定、用户透明、一维地址。消除外碎片
- 分段：逻辑单位、大小可变、用户可见、二维地址
- 快表(TLB)：命中1次访存，不命中2次（段页式3次）
- 虚拟存储：基于局部性原理。容量由地址线位数决定，非物理内存大小
- 页面置换：OPT（最优）→LRU→FIFO（有Belady异常）→Clock
- Belady异常：增加物理块反而增加缺页，仅FIFO存在

### 第5章 设备管理
- I/O控制演进：程序查询→中断→DMA→通道
- 通道：独立的I/O处理机，有通道指令集
- 设备分类：独占（打印机，静态分配）、共享（磁盘，动态分配）、虚拟（SPOOLing）
- SPOOLing：联机实现。三部分：输入/输出井、输入/输出进程、井管理程序
- I/O软件分层：用户层→设备无关层→设备驱动→中断处理
- 缓冲目的：缓和速度不匹配、减少中断频率、提高并行性
- 磁盘调度：FCFS→SSTF（最短寻道）→SCAN/电梯→C-SCAN（单向）
- 磁盘访问时间：寻道+旋转延迟+传输

## HTML 页面 CSS 变量体系

三页面统一使用以下 CSS 变量：

```css
:root {
  --bg: #f8f9fa;      /* 页面背景 */
  --c: #fff;          /* 卡片背景 */
  --t: #212529;       /* 正文颜色 */
  --m: #6c757d;       /* 辅助文字 */
  --b: #dee2e6;       /* 边框 */
  --a: #0d6efd;       /* 强调色/链接 */
  --s: #198754;       /* 成功/正确答案 */
  --warn: #dc3545;    /* 警告/易错/P0 */
}
```

- 答案使用原生 `<details><summary>`，禁止 JS 实现折叠
- P0用红色 `#dc3545`，P1用橙色 `#fd7e14`，P2用蓝色 `#0d6efd`
- 页面最大宽度 860px，居中
